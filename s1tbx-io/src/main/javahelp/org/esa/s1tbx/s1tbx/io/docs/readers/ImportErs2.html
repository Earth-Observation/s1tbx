Import ERS-2 SAR

Import ERS-2 Product in ENVISAT format

The ERS-1 and ERS-2 tandem operations demonstrated SAR interferometry for various applications and paved the way for the definition of new dedicated SAR interferometry missions.

Operational: to 2011-09-05

Frequency Band: C-band

Acquisition Modes:  Stripmap

Product Types:  

  • GEC Geocoded Ellipsoid Corrected
  • PRI Precision Image
  • SLC Single Look Complex

Polarizations: All modes are Single Pol (VV).

Opening products: Products can be opened by selecting the Volume file for CEOS format products or the *.E2 file for ENVISAT format products. Opening zipped products (*.zip) directly is also supported.
